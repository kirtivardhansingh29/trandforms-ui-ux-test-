# Transformer UI Landing Page

Standalone futuristic landing page built with plain **HTML + CSS + JavaScript**.

## Files

- `index.html` - page structure
- `style.css` - visual design, layout, motion, and responsive behavior
- `script.js` - live telemetry updates, motion handling, and CTA scroll behavior

## Run

Open `index.html` directly in a browser.

## Features

- Transformer-inspired mechanical core with layered depth visuals
- Glassmorphic telemetry cards and live system metrics
- Neon CTA and cinematic dark industrial background
- Keyboard focus styles and reduced-motion support
